# Useful programs and configs

## IMWheel for remapping keyboard and mouse buttons

Install
```shell
sudo apt install imwheel
```

Copy `.imwheel` to home directory.

and add it to auto load.

See also https://wiki.archlinux.org/title/IMWheel
